DCI

Что такое DCI?

  • Data - Что? (модели)
  • Context - Где? (use-кесы)
  • Interaction - Как? (специфика, роли)

Типичный пример DCI

      class MoneyTransfer # Context
        def initialize(source, destination) # Data
          @source = SourceAccount.new(source)
          @destination = DestinationAccount.new(destination)
        end
        def transfer(amount)
          # Interaction (используем роли)
        end
      end

MVC - частный случай DCI

      class ProjectsController
        # Context (controller)
        def show
          # Data (model)
          project = Project.find(params[:id])
          # Interaction Role (view)
          @presentable_project = ProjectPresenter.new(project)
        end
      end

Репозитарий (валидация, сохранение...)

      # Context
      class ProjectRepository
        def initialize(project) # Data
          @project = project
          @project.extend ProjectValidator, ProjectPersister
        end
        def save # Interaction (используем роли)
        end
      end
